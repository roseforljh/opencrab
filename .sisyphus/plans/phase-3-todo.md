# 阶段 3 TODO：前端联调前完整度提升

## 阶段目标

在后端还未开始前，把前端控制台自身的完整度继续补齐，达到可单独验收的程度。

## 待办清单

- [ ] 统一封装数据表格组件。
- [ ] 统一封装页面标题区组件。
- [ ] 统一封装统计卡片组件。
- [ ] 统一封装筛选条组件。
- [ ] 统一封装详情抽屉组件。
- [ ] 为所有页面补空状态。
- [ ] 为所有页面补加载状态。
- [ ] 为所有页面补错误状态。
- [ ] 用假数据模拟关键业务流程。
- [ ] 调整页面间距、层级、边框和密度细节。

## 完成标准

1. 前端组件复用关系清晰。
2. 六个页面在无后端情况下也能完整演示。
3. 视觉一致性达到可验收程度。

## 验证清单

1. 逐页检查空状态、加载状态、错误状态，预期都有统一表现。
2. 检查共享组件被多个页面复用，预期没有同类 UI 大量重复实现。
3. 检查页面细节，预期主次层级、边框、标题、按钮风格一致。
